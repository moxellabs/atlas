# MCP Prompts Module

The prompts module provides reusable grounding prompts.

## Responsibilities

- Answer from local docs.
- Onboard to a repo or module.
- Explain skill usage.
- Summarize modules.
- Compare docs.
- Register prompt definitions consistently.

## Invariants

Prompts should encourage tool-backed evidence and local corpus provenance rather than unsupported free-form answers.
