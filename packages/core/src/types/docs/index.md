# Core Types Module

The types module defines shared Atlas contracts.

## Responsibilities

- Repo and source adapter types.
- Canonical document, section, chunk, and summary types.
- Topology, skill, scope, and classification types.
- Retrieval query, hit, ambiguity, and planned context types.
- Provenance, freshness, and source change types.

## Invariants

Types should express domain boundaries clearly and avoid depending on runtime implementation packages.
